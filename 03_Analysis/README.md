# Statistical Analysis

This directory contains the statistical analyses performed in the study **FECGenAI-SE: A Framework for Evaluating the Influence of Culture and Artificial Intelligence Literacy on the Adoption of Generative Artificial Intelligence in Software Engineering**.

The analyses were conducted using **Jamovi** and include the complete project file together with exported statistical results.

## Files

### `FECGenAI-SE_Complete_Analysis.omv`

Complete Jamovi project containing all statistical analyses reported in the dissertation and manuscript, including:

- Descriptive statistics
- Participant profile analysis
- Reliability analysis (Cronbach's Alpha)
- Exploratory Factor Analysis (KMO, Bartlett's Test, and EFA)
- Chi-square tests of association
- Spearman correlation analyses

### `Spearman_Correlation_Matrix.xlsx`

Spearman correlation matrix calculated using the construct scores (mean values of the FECGenAI-SE constructs).

### `Spearman_Correlation_Matrix_All_Items.xlsx`

Spearman correlation matrix calculated using all questionnaire items individually (item-level analysis).

## Software

- Jamovi 2.x

## Reproducibility

Opening `FECGenAI-SE_Complete_Analysis.omv` reproduces all statistical analyses reported in the dissertation and the associated scientific publication. The exported correlation matrices are provided for quick reference and facilitate inspection of the statistical results without requiring Jamovi.
