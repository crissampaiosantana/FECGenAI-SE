# FECGenAI-SE

## Open Science Repository

This repository contains the research materials supporting the paper:

**A Framework for Evaluating the Influence of Culture and Artificial Intelligence Literacy on the Adoption of Generative Artificial Intelligence in Software Engineering**

The repository has been created to promote transparency, reproducibility, and reuse of the research in accordance with Open Science principles.

## Repository Structure

### 01_Questionnaire
Survey instrument used for data collection.

### 02_Dataset
Anonymized dataset and codebook describing all variables.

### 03_Analysis
Jamovi project file and statistical analysis outputs.

### 04_Framework
Conceptual framework and related figures.

### 05_Supplementary_Materials
Additional supporting documents and research materials.
